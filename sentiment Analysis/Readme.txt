graph.py : contains module for plotting graph.


How to compile and run:

Requirement:
Ubuntu
python
NLTK modules
numpy
scipy
scikit-learn
statistics module
tweepy
matplotlib

First compile Training.py file. Then run twitter_module and graph_module simultaneously.
